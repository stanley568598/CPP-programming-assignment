# RPG

## 介紹

這是一個簡單的 RPG (Role-Playing Game) 遊戲，可視為簡化版的電子桌遊系統。

## 功能

- [x] Character setting ( 角色建立 )
- [x] Move ( 角色移動 WASD )
- [x] Fight ( 戰鬥 )
- [x] Fate ( 命運 )
- [x] Scene change ( 場景變化 )
- [x] Goal ( 抵達終點 )
