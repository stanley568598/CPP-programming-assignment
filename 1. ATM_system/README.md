# ATM_system

## 介紹

這是一個簡單的 ATM (Automated Teller Machine) 系統，可視為簡化版的電子帳簿系統。

## 功能

測試帳號密碼 **( 環境預設 )**

> 測試帳號 1

    account number：12345
    PIN：54321
        
> 測試帳號 2
    
    account number：98765
    PIN：56789

- [x] Login ( 登入帳戶 )
- [x] View my balance ( 查看餘額 )
- [x] Withdraw cash ( 提領現金 )
- [x] Deposit funds ( 存入款項 )
- [x] Exit ( 離開系統 )
