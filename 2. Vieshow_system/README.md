# Vieshow_system

## 介紹

這是一個模擬 Vieshow 訂票功能的系統，可視為簡化版的電子售票系統。

## 功能

測試帳號密碼 **( 環境預設 )**

> 測試帳號 1

    email address：111
    password：111
        
> 測試帳號 2
    
    email address：222
    password：222

- [x] Sign In ( 登入 )
- [x] Show Account Information ( 查看/修改帳戶資訊 )
- [x] My Bookings ( 訂票紀錄 )
- [x] Buy Tickets ( 購票/劃位 )
- [x] Sign out ( 登出 )
